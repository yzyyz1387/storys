
# storys
https://story.nwafu.ac.cn  
random story  
一个flask框架的简单网站，无聊写的，为什么不用js等技术实现呢，因为还不会😎  
每次刷新网页都会随机一个故事，故事储存在根目录`story`文件下，每个故事格式为  
```
!story
【标题】
【作者】
【来源链接】
【来源】
【内容】
story!
```
